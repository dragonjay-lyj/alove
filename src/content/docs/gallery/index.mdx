<summary>隷奴X調教File 6</summary>

想着今年最后一次在户外进行裸体调教，如果天气晴朗，就按之前说好的，在山里的废弃仓库里玩吊缚和鞭打。

一到废弃仓库，就立刻准备用12毫米的麻绳进行吊缚。
